Build the latest development version

On a Linux machine with git and Java (alt least version 1.6) it's easy to compile WeaveAlign for yourself:
  1. Install Gradle (1.5 or more recent)
  2. Fetch sources from GitHub: git clone https://github.com/statalign/WeaveAlign.git
  3. Compile: in the project root (WeaveAlign directory) run the build script: ./build.sh


How to use

WeaveAlign expects a set of input alignments, and will build up an alignment network of the provided alignments. Then many new alignment paths (probably different from all the inputs) will be computed and a ,,best'' one will be selected. Run java -jar WeaveAlign.jar -help to get help.

Run the example (Linux)

You can run a small example bash script: ./run-example.sh
Have a look at the script itself, it's quite simple. Here's in brief what will happen:
  1. 500 alignments will be unpacked from a zip file to be used as the set of input alignments
  2. WeaveAlign will be called to build the network and select a good alignment
  3. The selected WeaveAlignment will be evaluated against a reference alignment, as well as the input alignments, so that we can see if we got a better result by combining the alignments
