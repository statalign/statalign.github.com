Or build the development version

On a Linux machine with git and Java (alt least version 1.6) it's easy to compile WeaveAlign for yourself:
  1. Install Gradle (1.5 or more recent)
  2. Fetch sources from GitHub: git clone https://github.com/statalign/WeaveAlign.git
  3. Compile: in the project root (WeaveAlign directory) run the build scipt: ./build.sh
